bc4e7b8d-f5fe-5d73-a71f-92e3690744435.2.1 Keyword search5.2.1 关键词搜索
6ef648cd-298f-5056-8dad-7198df3f8881Keyword search is the most traditional search method, evaluating relevance by calculating the frequency and importance of words contained in documents.关键词搜索是最传统的搜索方法，通过计算文档中包含的词语频率和重要性来评估相关性。
26b28f40-199c-5f12-96e5-be3a13aed867Representative algorithms include BM25 and TF-IDF.代表性算法包括BM25和TF-IDF。
baf35350-eb6e-5f48-9f3c-40473bc0699cTF-IDF (Term Frequency-Inverse Document Frequency) combines two metrics.TF-IDF（词频-逆文档频率）结合了两种度量指标。
c084e4bc-1a39-593e-ae5d-042dd5107ec5It multiplies how frequently a specific word appears within a document (TF) by how rare that word is across the entire document collection (IDF).它将特定词语在文档中出现的频率（TF）乘以该词语在整个文档集合中的稀有程度（IDF）。
01acfa5d-d793-5cbf-8a84-3bcce140d0f4Words like "the" that appear everywhere have low importance, while words that appear intensively only in specific documents have high importance.像“the”这类随处可见的词语重要性较低，而仅密集出现在特定文档中的词语则具有较高的重要性。
97a09298-79c1-51b6-b53c-e7acd9347efdBM25 is an improved algorithm over TF-IDF that provides more accurate results by adjusting for document length and reflecting the saturation effect of word frequency.BM25是TF-IDF的改进算法，通过调整文档长度并反映词频的饱和效应，能提供更准确的结果。
920861ca-5296-5224-8673-5d8ac383bb12Keyword search is most effective when finding exact words or phrases.关键词搜索在查找确切单词或短语时最为有效。
238f29c3-dfb9-5384-8e89-a1d1b9fa078dThere's no method faster or more accurate than keyword search when searching for a function name like get_user_by_id, finding error code 404, or checking a specific configuration value like MAX_CONNECTIONS.在搜索函数名称（如get_user_by_id）、查找错误代码404或检查特定配置值（如MAX_CONNECTIONS）时，没有比关键词搜索更快或更准确的方法。
e2e2177e-99ff-5c0d-b785-fca23de2cfa2As a technology proven over decades, it guarantees fast search speeds even on large-scale datasets.作为一项经过数十年验证的技术，它即使在大型数据集上也能保证快速的搜索速度。
38b7426b-fa50-5982-ace4-98a448afbfccHowever, keyword search has fundamental limitations.然而，关键词搜索存在根本性局限。
1cecb101-3ec1-595e-a5ff-ea4f64c957e9It cannot handle synonyms or similar expressions.它无法处理同义词或类似表达。
4c93eebe-cf10-5ec4-a9e7-6aca30ec6aa5Figure 5.4 illustrates this problem.图5.4展示了这个问题。
f40ed885-0c8a-51ab-bc17-9880878615c5"Sofa" and "couch" refer to the same furniture, but keyword search doesn't know these two are related.“沙发”和“长沙发”指的是同一种家具，但关键词搜索不知道这两者相关。
607c8878-b073-54d2-a6d5-73e0372c826fSearching for "sofa" won't find documents containing "couch." Without building a separate synonym dictionary, this semantic connection cannot be automatically recognized.搜索“沙发”不会找到包含“长沙发”的文档。如果不构建单独的同义词词典，这种语义关联就无法被自动识别。

b786d050-0535-5e60-8d94-80093665f945Vector search is a method that quantifies the meaning of text and calculates similarity.向量搜索是一种量化文本含义并计算相似度的方法。
109d3d25-cde0-53dc-9c11-55a7953f257fFirst, it uses an embedding model to transform text into a point in a high-dimensional vector space.首先，它使用嵌入模型将文本转换为高维向量空间中的一个点。
